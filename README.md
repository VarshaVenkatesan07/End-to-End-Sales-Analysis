# End-to-End Sales Analytics Dashboard

A data-driven sales analytics project developed using **Power BI, Python, and Pandas** to analyze Superstore sales data, track key business KPIs, visualize sales performance, and generate actionable insights for decision-making.

### Technologies Used

* Power BI
* Python
* Pandas
* Excel

### Key Highlights

* Data Cleaning & Analysis
* KPI Tracking
* Sales Trend Visualization
* Business Insights Dashboard
